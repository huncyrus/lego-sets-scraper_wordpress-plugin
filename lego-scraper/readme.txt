=== Lego Sets scraper ===

== Description ==
Lego sets detail scraper from DK based online shop. Fetching raw html, using DOMXML & DOMXPath.

== Requirements ==
 - PHP 5x
 - DOMXML module
 - MySQL/MariaDB
 - Wordpress 4.x (4.4 preferably)
 - cUrl/libCurl

== Install ==
 - Copy into the wordpress /wp-content/plugins directory
 - Enable it @ WP admin
